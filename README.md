# homebrew-visr

Homebrew tap for [visr](https://github.com/sourishkrout/visr) — Agent UX testing.

## Installation

```bash
brew tap sourishkrout/visr
brew install visr
```

## Usage

See the [main visr repository](https://github.com/sourishkrout/visr) for documentation.

## Updating

```bash
brew upgrade visr
```
